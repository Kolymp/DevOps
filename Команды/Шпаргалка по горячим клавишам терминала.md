```
# Навигация
Ctrl + A          # Начало строки
Ctrl + E          # Конец строки
Ctrl + U          # Удалить от курсора до начала
Ctrl + K          # Удалить от курсора до конца
Ctrl + W          # Удалить слово перед курсором
Ctrl + L          # Очистить экран (как clear)
Ctrl + R          # Поиск по истории команд
Ctrl + C          # Прервать команду
Ctrl + Z          # Приостановить процесс (в фон)
Ctrl + D          # Выход (EOF)

# История
!!                # Повторить последнюю команду
!$                # Последний аргумент предыдущей команды
!^                # Первый аргумент предыдущей команды
!:1-3             # Аргументы 1-3 предыдущей команды
```
